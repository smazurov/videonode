# V4L2 Camera Detector

This project provides a command-line tool to detect and query V4L2 camera devices on Linux systems. It enumerates available devices, their supported formats, resolutions, framerates, and controls.

## Inspiration

This project was inspired by and derived from the V4L2 plugin code found in OBS Studio.
