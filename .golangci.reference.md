# golangci-lint Configuration Reference

## Documentation Linters

| Linter | Purpose |
|--------|---------|
| godot | Comments must end with period |
| revive/exported | Require comments on exported symbols |
| revive/package-comments | Require package-level docs |
| errname | Error naming (ErrFoo, FooError) |
| staticcheck | Style rules (ST*) included by default |

## Re-enabled Exclusions

EXC0012-EXC0015 re-enabled to enforce documentation warnings.

## Project Initialisms

ID, URL, API, HTTP, JSON, XML, SSE, SBC, FFmpeg, V4L2, HDMI, RTSP, WebRTC, LED, USB, OBS, MPP, ALSA

## Running Locally

```bash
golangci-lint run ./...          # full check
golangci-lint run --fix ./...    # auto-fix
```
